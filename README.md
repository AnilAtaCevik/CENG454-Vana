# Chopper-Strike: Global Operations
![Logo](Logo.svg)

## Game Description
**Chopper-Strike: Global Operations** is an action-oriented 2.5D helicopter platformer. The player takes control of a military combat helicopter, navigating through challenging platforming environments and engaging in tactical combat to destroy enemy units. 

A core element of the gameplay loop is strict resource management; players must carefully calculate their flight maneuvers, overcome environmental obstacles, and execute attacks while efficiently managing a limited fuel supply. 

**Objective & Conditions:** 
The primary objective is to successfully navigate through the hostile zones and reach the end of the stage. 
- **Win Condition:** Reaching the extraction point / finishing line at the end of the level.
- **Lose Condition:** The game is lost if the helicopter is destroyed by enemy fire or environmental hazards, or if the fuel tank is completely depleted mid-flight.

## Setup
### 1. Playing the Game (Windows x64 Build)
To play the presentation-ready build of the game:
1. Download or clone this repository to your local machine.
2. Navigate to the `Builds` folder located in the root directory of the repository.
3. Double-click the executable file (e.g., `ChopperStrike.exe`) to launch the game.
*(Note: Do not move the `.exe` file outside of the `Build` folder. It must stay in the same directory as its associated `_Data` folder to run properly.)*

### 2. Opening the Project in Unity (Source Code Backup)
If you need to inspect the source code, check the patterns, or if the build fails to run, follow these steps to open the project:
1. Open **Unity Hub**.
2. Click on the **Add** (or **Open**) button and select the root folder of this cloned repository.
3. Make sure to open the project using **Unity version 6000.3.9f1** to avoid any compatibility issues or pipeline errors.
4. Once the editor is fully loaded, navigate to the `Assets/Scenes` folder and open the main level scene to explore the project.
